# RayRhythm

RayRhythm is a rayman themed vertical scolling rhythm games (VSRG). Inspired by games like osu!, quaver, etterna and ddr. Rayman is a video game charecter from late 90s and the 21st century, featured in games like Rayman 2, Rayman Origins, Rayman Legends and many more.

Made with PyGame library: https://www.pygame.org/
